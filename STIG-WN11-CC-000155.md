# STIG: WN11-CC-000155  
Solicited Remote Assistance must be disabled

---

## Identifiers
- STIG ID: WN11-CC-000155  
- Platform: Windows 11 Pro 64-bit (Azure VM)

---

## Finding

Solicited Remote Assistance was enabled on the system, allowing users to request remote support sessions.

---

## Security Impact

If Solicited Remote Assistance remains enabled:

- Remote connections may be established from external hosts.  
- Increased exposure to social engineering and unauthorized access attempts.  
- Expanded attack surface for remote exploitation.  
- System remains out of compliance with STIG security requirements.  

---

## Remediation and Validation

Disabled Solicited Remote Assistance by enforcing the required policy-backed registry configuration and verified the change using PowerShell.

### PowerShell Remediation Script

```powershell
# Create policy path if it does not exist
New-Item -Path "HKLM:\SOFTWARE\Policies\Microsoft\Windows NT\Terminal Services" -Force | Out-Null

# Disable Solicited Remote Assistance
Set-ItemProperty `
    -Path "HKLM:\SOFTWARE\Policies\Microsoft\Windows NT\Terminal Services" `
    -Name "fAllowToGetHelp" `
    -Type DWord `
    -Value 0
