# STIG: WN11-CC-000155  
Solicited Remote Assistance must be disabled

---

## Identifiers
- STIG ID: WN11-CC-000155  

---

## System Context

 Windows 11 Pro 64-bit (Azure VM)
 
---

## Finding

The system configuration allowed users to request Solicited Remote Assistance sessions.

When enabled, this feature permits a user to invite another system to establish a remote support connection.

---

## Risk Analysis

Allowing Solicited Remote Assistance introduces the following risks:

- Potential for unauthorized remote connectivity.

- Elevated risk of social engineeringâ€“driven compromise.

- Increased remote attack surface.

---

## Remediation Action

The configuration was corrected by disabling Solicited Remote Assistance using a policy-backed registry setting.

### PowerShell Remediation Command

```powershell
# Ensure required registry path exists
New-Item -Path "HKLM:\SOFTWARE\Policies\Microsoft\Windows NT\Terminal Services" -Force | Out-Null

# Enforce policy to disable Solicited Remote Assistance
Set-ItemProperty `
    -Path "HKLM:\SOFTWARE\Policies\Microsoft\Windows NT\Terminal Services" `
    -Name "fAllowToGetHelp" `
    -Type DWord `
    -Value 0

Get-ItemProperty `
    -Path "HKLM:\SOFTWARE\Policies\Microsoft\Windows NT\Terminal Services" `
    -Name "fAllowToGetHelp"
