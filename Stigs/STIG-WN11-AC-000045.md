# STIG: WN11-AC-000045  
Reversible password encryption must be disabled

---

## Identifiers
- STIG ID: WN11-AC-000045
- Platform: Windows 11 Pro 64-bit (Azure VM)

---

## Finding

The system was configured with:

ClearTextPassword = 1

This setting enables storage of passwords using reversible encryption.

---

## Security Impact

If left enabled:

- Stored credentials may be decrypted if system access is compromised.
- Password confidentiality protections are weakened.
- System remains non-compliant with DISA STIG requirements.

---

## Remediation

Configured the system to disable reversible password encryption by setting:

ClearTextPassword = 0

### PowerShell Remediation Commands

```powershell
# Export current local security policy
secedit /export /cfg C:\secpol.cfg

# Modify the exported file:
# Set ClearTextPassword = 0

# Re-import the updated policy
secedit /configure /db C:\Windows\Security\Local.sdb /cfg C:\secpol.cfg /areas SECURITYPOLICY
```

---

## Validation

### PowerShell Validation Command

```powershell
secedit /export /cfg C:\verify.cfg
Select-String -Path C:\verify.cfg -Pattern "ClearTextPassword"
```

Expected result:

ClearTextPassword = 0
