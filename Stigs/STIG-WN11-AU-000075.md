# STIG: WN11-AU-000075  
The system must be configured to audit Logon/Logoff â€“ Logon successes

---

## Identifiers
- STIG ID: WN11-AU-000075
- Platform: Windows 11 Pro 64-bit (Azure VM)

---

## Finding

The system was not configured to audit successful logon events.

Audit policy for the "Logon" subcategory did not include Success.

---

## Security Impact

If left unconfigured:

- Successful logon activity is not recorded.
- Reduced visibility into account usage.
- Impaired detection of unauthorized access.
- System remains non-compliant with DISA STIG auditing requirements.

---

## Remediation

Configured the audit policy to enable logging of successful logon events.

### PowerShell Remediation Command

```powershell
auditpol /set /subcategory:"Logon" /success:enable
```

---

## Validation

### PowerShell Validation Command

```powershell
auditpol /get /subcategory:"Logon"
```

Expected result:

Logon  
  Success and Failure   (or at minimum: Success enabled)
