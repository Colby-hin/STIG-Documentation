# STIG: WN11-CC-000110  
HTTP-based printing must be disabled

---

## Identifiers
- STIG ID: WN11-CC-000110
- Platform: Windows 11 Pro 64-bit (Azure VM)

---

## Finding

The system allowed HTTP-based printer communication.

The registry setting controlling HTTP printing was not configured to disable the feature.

---

## Security Impact

If HTTP printing remains enabled:

- Sensitive documents could be intercepted.
- Increased exposure to network-based attacks.
- System remains non-compliant with DISA STIG requirements.

---

## Remediation

Configured the policy to disable HTTP-based printing via registry enforcement.

### PowerShell Remediation Script

```powershell
# Ensure policy path exists
New-Item -Path "HKLM:\SOFTWARE\Policies\Microsoft\Windows NT\Printers" -Force | Out-Null

# Disable HTTP Printing
Set-ItemProperty `
    -Path "HKLM:\SOFTWARE\Policies\Microsoft\Windows NT\Printers" `
    -Name "DisableHTTPPrinting" `
    -Value 1 `
    -Type DWord
```

---

## Validation

### PowerShell Validation Command

```powershell
Get-ItemProperty `
    -Path "HKLM:\SOFTWARE\Policies\Microsoft\Windows NT\Printers" `
    -Name "DisableHTTPPrinting"
```

Expected result:

DisableHTTPPrinting = 1
